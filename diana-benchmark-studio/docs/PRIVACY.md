# Privacy boundary

Programs are generated in this browser and run on the researcher's machine. The Studio does not upload datasets, configurations, or generated programs. It uses no analytics, tracking, remote fonts, third-party runtime scripts, backend APIs, or server-side Notebook execution.

Configuration import accepts bounded Studio JSON/YAML and retains it in browser memory. Generated kits use compiler-fixed relative data paths and contain deterministic synthetic examples only. Licensed mcPHASES rows, participant IDs, sample IDs, folds, predictions, and private paths are never embedded in the Hormonbench preset.
